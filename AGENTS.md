# AGENTS.md

## Project Context

This is the Bihar Government Frontend application repository. Keep changes focused on the user's request and preserve existing project conventions.

Start with `README.md` for local setup and environment variables.

## Key Files

- `src/`: frontend application source code.
- `src/lib/axios.js`: standard Axios HTTP client configuration.
- `vite.config.js`: Vite configuration setup.
- `.env`: local environment configuration values; never commit secrets.

## Working Notes

- Use `npm run dev` for local development.
- Use `npm run build` to create a production bundle.
- Run the relevant checks from `package.json` before finishing code changes.
